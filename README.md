## 🚀 Project Overview

**Rural ERides Go** is a production-grade React, Vite, and Capacitor mobile application engineered and heavily field-tested in Stigler, Oklahoma. Designed exclusively for electric personal vehicle (PEV) enthusiasts, this app bypasses closed-ecosystem restrictions to deliver high-precision telemetry monitoring, real-time weather tracking, and seamless multi-device coordination.

This application officially launched in tandem with the **150 Subscriber Milestone** on the Rural Erides channel, celebrating a growing community of rural riders and open-source tech enthusiasts!

## ✨ Core Architecture & Features

*   **Live GPS Telemetry HUD:** Real-time speedometers, physics-based incline/decline detection, G-sensor braking force calculations, and cumulative mileage tracking.
*   **Telemetry Data Export:** Built-in CSV and GPX log export functions designed to generate transparent speedometers and HUD overlays for CapCut video edits.
*   **Dual-Device Rider Radar:** Sub-meter precision tracking enabling multi-device fleet monitoring between mobile phones and field tablets with built-in privacy protection (powered by Supabase auth).
*   **12-Point Atmospherics Matrix:** Real-time weather data tracking wind gust vectors, UV solar indexes, surface soil temperatures, dew points, and dynamic ride safety ratings to prepare for unpredictable rural weather patterns.
*   **AI Mechanical Copilot:** Secure API routing middleware (`CoPilotService.ts`) for optical inspection, battery/brake diagnostics, and native Text-to-Speech support.
*   **Embedded Audio Deck:** Multi-page media integration pre-loaded with tailored ride audio ranging from classical piano and theatre organ consoles to vintage mobile phone stock ringtones.
