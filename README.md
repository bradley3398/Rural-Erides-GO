## 🚀 Project Overview
**Rural ERides Go** is a production-grade React, Vite, and Capacitor mobile application engineered and field-tested in Stigler, Oklahoma by **Lord Bradley Callison** (`@bradleycallison`). Built specifically for electric personal vehicle (PEV) enthusiasts, it delivers high-precision telemetry monitoring, weather tracking, and multi-device coordination without closed-ecosystem restrictions.

---

## 🚲 Active Personal Fleet & Hardware Setup
* **Electric Vehicle Fleet:** Aostirmotor A20 Folding Bike, Geemax E-Trike, isinwheel H7 Pro Scooter, and Engwe Y600.
* **Primary Field Rig:** Samsung Galaxy S25 Plus mobile phone running high-precision GPS tracking, video capture, and multi-device Rider Radar.

---

## ✨ Core Architecture & Features
* **Live GPS Telemetry HUD:** Real-time speedometers, physics-based incline/decline detection, G-sensor braking force calculations, and cumulative mileage tracking.
* **Telemetry Data Export:** Built-in CSV and GPX log export functions designed to generate transparent speedometers and HUD overlays for CapCut video edits.
* **Dual-Device Rider Radar:** Sub-meter precision tracking enabling multi-device fleet monitoring between mobile phones and field tablets with built-in privacy protection.
* **12-Point Atmospherics Matrix:** Real-time weather data tracking wind gust vectors, UV solar indexes, surface soil temperatures, dew points, and dynamic ride safety ratings.
* **AI Mechanical Copilot:** Secure API routing middleware (`CoPilotService.ts`) for optical inspection, battery/brake diagnostics, and native Text-to-Speech support.
* **Embedded Audio Deck:** Multi-page media integration pre-loaded with tailored ride audio ranging from classical piano and theatre organ consoles to synthwave and phonk mixes.
